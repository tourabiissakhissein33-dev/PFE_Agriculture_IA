# 🌾 Agro-IA Tchad — Cadre Décisionnel Basé sur l'IA

**Projet de Fin d'Études — Licence 3 Informatique**  
École Nationale Supérieure des TIC (ENASTIC) — 2025/2026

## 📌 Description
Système d'aide à la décision agricole basé sur l'intelligence artificielle
pour l'optimisation du suivi parcellaire au Tchad.

## 🤖 Modules IA
| Module | Algorithme | Accuracy |
|--------|-----------|----------|
| 💧 Irrigation | Random Forest | 78.51% |
| 🌿 Fertilisation | XGBoost | 95.42% |
| 🔬 Maladies | CNN MobileNetV2 | 96.61% |
